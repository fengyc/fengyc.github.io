git常用命令

Permalink: 2014-08-24 17:40:00 by Yingcai FENG in 博客 tags: git

下载远程仓库

git clone <URL>

在本地工作目录指定远程仓库

git remote add <URL>

查看远程仓库

当git仓库clone到本地之后，如果忘记了具体的远程仓库地址，可通过以下的命令查看： git remove -v
